# 1.'Data Jobs' Satisfaction Analysis
## Problem Statement
Organizations often struggle to understand employee satisfaction levels across different roles, departments, and work-life factors. This project aims to analyze survey data to identify trends, risks, and key areas for improvement.
## Objectives
- Analyze employee satisfaction levels
- Compare satisfaction across key factors
- Provide insights to support data-driven decisions
## Dataset
- Source: Real life employee survey data
- Number of records: 600+ responses
- Key columns: Unique ID, Job Title, Salary Satisfaction Level,Work-Life Balance Satisfaction Level, Average Yearly Income
## Tools Used
- Excel (data cleaning and manipulation)
- Power BI
- Power Query (for data cleaning & transformation)
## 
## Solutions & Recommendations
- Introduce flexible schedules or workload redistribution for roles where low satisfaction is driven by long working hours
- Introduce structured career growth plans for mid-level employees.
- Apply targeted engagement strategies to high-risk roles.
